# 🏆 Site Jogos Internos IFAL

Site desenvolvido como projeto acadêmico para a disciplina de Programação Web, com o objetivo de apresentar as informações dos Jogos Internos da instituição.

---

## 📋 Sobre o Projeto

O site reúne todas as informações dos Jogos Internos do IFAL [ano] em um único lugar: programação dos jogos, modalidades esportivas, equipes participantes, galeria de fotos e histórico de edições anteriores.

---

## 🗂️ Páginas

| Arquivo | Página | Descrição |
|---|---|---|
| `index.html` | Home | Página principal com banner, resumo e destaques do evento |
| `programacao.html` | Programação | Tabela com dias, horários, esportes e locais |
| `modalidades.html` | Modalidades | Cards com os esportes e suas descrições |
| `equipes.html` | Equipes | Turmas participantes com cores e símbolos |
| `galeria.html` | Galeria | Fotos das edições dos jogos |
| `contato.html` | Contato | Formulário de contato |
| `historico.html` | Histórico | Campeões e destaques de edições anteriores |

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estruturação do conteúdo
- **CSS3** — estilização das páginas
- **Flexbox** — organização e alinhamento dos layouts
- **Google Fonts** — tipografia externa

## 🚀 Como Executar

1. Clone ou baixe o repositório:
   ```bash

   ```

2. Abra o arquivo `index.html` diretamente no navegador, ou utilize uma extensão como o **Live Server** (VS Code) para melhor experiência de desenvolvimento.
